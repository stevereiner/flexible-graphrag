üöÄ Quick Start

  1. Set VECTOR_DB=lancedb in your .env file
  2. Configure the database URI (default: ./lancedb)
  3. Start Flexible GraphRAG to create the database
  4. Access LanceDB Viewer at http://localhost:3005
  5. Browse tables, run queries, and manage your data!
