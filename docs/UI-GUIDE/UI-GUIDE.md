# UI Guide — Tab Overview

The Flexible GraphRAG UI provides a 4-tab interface for document processing and querying.
Available as React, Angular, and Vue frontends — all with the same capabilities.

| Tab | Purpose |
|---|---|
| **[Sources](UI-TAB-SOURCES.md)** | Configure your data source and select files |
| **[Processing](UI-TAB-PROCESSING.md)** | Start ingestion and monitor progress |
| **[Hybrid Search](UI-TAB-SEARCH.md)** | Hybrid Search and AI Query |
| **[AI Chat](UI-TAB-CHAT.md)** | Conversational AI over your documents |
